# faceTrans
仿照茄子快传 ，实现了一种通过Wi-Fi热点，免流量app之间传递数据的功能，主要的原理是通过Wi-Fi热点，然后利用socket通信实现数据的传递

目前传递的方式有2中，一个是app之间的传递 ，另外一种是通过网页传递，两种方法都已经实现
